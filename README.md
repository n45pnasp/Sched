# Penempatan Posisi Jaga

Aplikasi web lokal berbasis Python (Flask) untuk mengolah jadwal
penempatan posisi jaga dari file `sched.xlsx`.

## Fitur
- Upload Excel jadwal (sched.xlsx)
- Proses otomatis sesuai logika operasional
- Output Excel hasil + sheet AUDIT
- Logika diproses penuh di Python (stabil & deterministik)

## Requirement
- Python 3.9+
- pip

## Install
```bash
pip install -r requirements.txt
